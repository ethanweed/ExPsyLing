#### When intuitions clash with logic

|                     | conclusion feels true | conclusion feels false |
|---------------------|-----------------------|------------------------|
| argument is valid   | 92% say "valid"       | 46% say "valid"        |
| argument is invalid | 92% say "valid"       | 8% say "valid"         |
